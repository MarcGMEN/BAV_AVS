Vous enregistrez un dépôt pour :
  • --obj_type-- --obj_public-- avec une pratique : --obj_pratique--
  • --obj_marque-- - --obj_modele-- de couleur : --obj_couleur--
  • Description : "--obj_description--"
  • PRIX : --obj_prix_depot--

  • Vous : --cli_nom-- (--cli_telephone--)
    • --cli_adresse-- --cli_adresse1--
    • --cli_code_postal-- --cli_ville--

Vous allez recevoir un mel à l'adresse --cli_emel-- pour confirmer ce dépot.
Une fois cette confirmation effectuée, vous recevrez la fiche de dépôt ainsi que les instructions de dépôt.
